# Jobs

Jobs are units of work for a Site.

## Key Fields

- Site
- Optional Job Template
- Optional Work Programme
- Title and description
- Estimated duration
- Priority
- Status
- Completed date
- Closeout note

## Statuses

- Unassigned
- Assigned
- Completed
- Cancelled

Assigned Jobs are linked to a Site Visit. Completed Jobs require a completed
date. Cancelled Jobs require a closeout note.

## Closeout Ownership

When a Job is assigned to a Trip, normal Job editing does not own status,
completed date, or closeout note. Those fields are resolved by Trip closeout.
