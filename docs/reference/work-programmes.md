# Work Programmes

Work Programmes are dated batches of Jobs with a shared planning period or due
date.

## Key Fields

- Name
- Start date
- End date
- Description

Dates can be left unset while the programme is still being shaped. If both dates
are set, the end date cannot be before the start date.

## Relationships

Jobs can optionally belong to a Work Programme. A Job can exist without one when
the work is not part of a dated batch.

Object History records Work Programme changes.
